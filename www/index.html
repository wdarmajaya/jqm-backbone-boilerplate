This is a boilerplate web application based on the example technique suggested by jQuery Mobile (JQM) to utilize Backbone.js and Require.js to help structure the code. This app is responsive and can be accessed from both desktop browsers and mobile browsers, and in addition, it can be packaged as native apps (ios and android) using Phonegap.

Frameworks used to develop this boilerplate:

jQuery Mobile is a HTML5-based user interface system designed to make responsive web sites and apps that are accessible on all smartphone, tablet and desktop devices.

Backbone.js is a great client-side MV* JavaScript framework that provides structure to JavaScript applications by providing View, Model, Collection, Router, and Event class objects.

Require.js is an AMD (Asynchronous Module Definition) script loader that asynchronously loads your JavaScript to improve page load performance, while also helping with script dependency managagement and allowing you to organize your JavaScript into self contained modules (files).

Handlebars provides the power necessary to let you build semantic templates effectively with no frustration.

Bootstrap is the most popular front-end framework for developing responsive, mobile first projects on the web.

PhoneGap is a free and open source framework that allows you to create mobile apps using standardized web APIs for the platforms you care about.
