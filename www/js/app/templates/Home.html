About This Web App

This is a boilerplate web application based on the example technique suggested by jQuery Mobile (JQM) to utilize Backbone.js and Require.js with JQM to help structure the code. This app is designed based on JQuery Mobile demo page


Additional Features

Responsive

This web app can be accessed from desktop and mobile devices. The responsiveness is achieved by using CSS media query, Bootstrap responsive utilities css classes (.visible-xs, .hidden-xs, etc), and PhoneGap device plugin (window.PhoneGap) for showing and hiding content by device.

  • On desktop view, menu panel is displayed on left side.
  • On mobile view, menu panel is hidden, menu icon is added to header, and header is fixed on top of screen.
  • On PhoneGap, external links are opened within the app (using inappbrowser plugin) instead of phone’s browser so user doesn’t have to leave the app.


PhoneGap

This web app can be packaged as iOS and Android app using PhoneGap. We subscribe to PhoneGap build service (build.phonegap.com) and to build an app just need to zip www folder and import it to the build service, or connect the build service to a GitHub repository.

Frameworks/Libraries used

jQuery Mobile is a HTML5-based user interface system designed to make responsive web sites and apps that are accessible on all smartphone, tablet and desktop devices.

Backbone.js is a great client-side MV* JavaScript framework that provides structure to JavaScript applications by providing View, Model, Collection, Router, and Event class objects.

Require.js is an AMD (Asynchronous Module Definition) script loader that asynchronously loads your JavaScript to improve page load performance, while also helping with script dependency managagement and allowing you to organize your JavaScript into self contained modules (files).

Handlebars provides the power necessary to let you build semantic templates effectively with no frustration.

Bootstrap is the most popular front-end framework for developing responsive, mobile first projects on the web.

PhoneGap is a free and open source framework that allows you to create mobile apps using standardized web APIs for the platforms you care about.

fastclick. Polyfill to remove click delays on browsers with touch UIs.

alertify.js. Unobstrusive notification messages.
